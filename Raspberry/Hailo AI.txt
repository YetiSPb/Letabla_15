Включение PCIe Gen 3.0 и модуля Hailo AI

---------------------------------------------------------------------------------------------
To enable PCIe Gen 3.0 speeds, add the following line to 

sudo nano /boot/firmware/config.txt
изменить или добавить
dtparam=pciex1_gen=3

sudo reboot

Настройка модуля AI
https://www.raspberrypi.com/documentation/computers/ai.html
Работает пока только на debian bookworm
Камера OV5647 

Обновление системы 
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install -y build-essential
#sudo apt-get install -y raspberrypi-kernel-headers


Обновить загрузчик, если Ruspbery старый
sudo raspi-config 

sudo reboot

Установить Hailo
sudo apt install hailo-all

sudo reboot
